# Arbitrageur Extension

This extensions is designed to work alongside [Darkpool](../scripts/Darkpool.md)'s HTML site generator. It adds a download button for the [Arbitrageur](../scripts/Arbitrageur.md)-generated Jupyter notebook to the bundle page via linked data.

## Settings

|Arg|Variable|Description|Notes|
|---|---|---|---|
|--arbitrageur-darkpool|LIBMM_ARBITRAGEUR_DARKPOOL|URL to Darkpool base directory|omit ending slash|
|--arbitrageur-directory|LIBMM_ARBITRAGEUR_DIRECTORY|Directory of the Arbitrageur notebooks|omit ending slash|
